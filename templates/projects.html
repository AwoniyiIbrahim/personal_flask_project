Musings & Ink

Musings & Ink is a personal blogging platform designed to showcase thoughts, experiences, and creative writing in a clean, minimalist interface. This project emphasizes simplicity, readability, and user-friendly navigation.

  • Responsive layout for desktop and mobile devices.
  • Dedicated section for individual blog posts.
  • Easy navigation with Home, About, Contact menu.
  • Modern design aesthetics inspired by Start Bootstrap templates.

Technology Used

Built using Flask for dynamic content and routing, demonstrating skills in web development and responsive design.

Purpose

Serves as a portfolio piece and a functional platform for creative expression.
